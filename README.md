## рҹ“ҡ Card Game Glossary

Some terms you'll see across these rules:

| Term | Meaning |
|------|---------|
| **Trick** | One round where each player plays one card; highest card wins |
| **Trump** | A suit that beats all other suits for the round |
| **Meld** | A combination of cards (e.g. three of a kind) laid down for points |
| **Shedding** | Game type where the goal is to get rid of all your cards |
| **Hand** | The cards you currently hold |
| **Stock** | The face-down pile you draw from |
| **Discard** | The face-up pile where played cards go |
| **Rank** | The number/value of a card (2 through Ace) |
| **Suit** | вҷ пёҸ Spades, вҷҘпёҸ Hearts, вҷҰпёҸ Diamonds, вҷЈпёҸ Clubs |
