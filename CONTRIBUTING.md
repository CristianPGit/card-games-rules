# 🤝 Contributing

Thanks for wanting to add to the collection!

## Adding a new game

1. Create a new file in `games/` named `your-game.md` (lowercase, kebab-case).
2. Follow the template below.
3. Add a row to the games table in [`README.md`](README.md).
4. Open a PR.

## File template

```markdown
# 🎴 Game Name

> One-line hook describing what makes the game fun.

<div align="center">

| 👥 Players | 🃏 Deck | ⏱️ Time | ⭐ Difficulty |
|:----------:|:------:|:------:|:------------:|
| X–Y | 52 cards | N min | Easy/Medium/Hard |

</div>

---

## 🎯 Goal
## 🃏 Setup
## 🎮 How to Play
## 🏆 Scoring
## 💡 Strategy Tips
## ⚠️ Common Mistakes

---

[← Back to all games](../README.md)
```

## Style guide

- ✅ **Plain language** — explain like you're teaching a friend.
- ✅ **Visual aids** — tables, ASCII card layouts, emoji headers.
- ✅ **Self-contained** — a reader shouldn't need to look anything up.
- ❌ **No paragraphs of theory** — keep it scannable.
- ❌ **No regional jargon** without explanation.

## Fixing errors

Found a wrong rule or a typo? Open a PR with the fix and a one-line note on where the correction came from (a rulebook, a wiki link, etc.).
